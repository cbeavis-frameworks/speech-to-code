# SpeechToCode

A macOS application that enables voice control for coding environments, featuring a streamlined Node.js installation process.

## Features

- Simple Node.js installer with automated download and configuration
- Proper verification of npm functionality
- User-friendly installation progress tracking
- Persistent installation state management using SwiftData
- Clean architecture with separation of concerns

## Requirements

- macOS 13.0+
- Xcode 15.0+
- Swift 5.9+

## Installation

1. Clone the repository
2. Open the Xcode project
3. Build and run the application

## Project Structure

- **Models**: Data models for tracking installation state
- **Services**: Core services for Node.js installation and process management
- **Views**: SwiftUI interface components
- **Utilities**: Helper classes and extensions
